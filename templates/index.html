WorkCalendarClient

Ваш надёжный справочник по рабочим дням

Что это?

Сервис, который точно знает, какой день — рабочий, а какой — выходной или праздничный.

Как это работает?

  • собирает данные из официальных источников и календарей;
  • проверяет несколько источников, если один недоступен;
  • хранит информацию в кэше для быстрых ответов;
  • предоставляет данные через простой API.

Кому пригодится?

  • разработчикам CRM и ERP‑систем;
  • командам, планирующим задачи и отпуска;
  • финансовым сервисам для расчёта дат платежей;
  • всем, кто хочет избежать ошибок из‑за праздников.

Быстрый старт

  1. Настройте окружение через .env-файл.
  2. Установите зависимости:
    pip install -r requirements.txt
  3. Запустите сервер:
    uvicorn app.main:app --host 0.0.0.0 --port 8000
  4. Отправьте запрос:
    curl "http://localhost:8000/is-working-day/2025-01-10"
